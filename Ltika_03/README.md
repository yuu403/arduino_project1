# 🚦 信号機制御システム

## 📌 概要
Arduinoを用いて、赤・黄・緑のLEDを制御し、信号機の動作を再現したプロジェクトです。  
状態遷移と時間制御を組み合わせた実践的なプログラムを実装しています。

---

## ⚙️ 動作
以下の順番でLEDが切り替わります：

- 🔴 赤（停止）
- 🟢 緑（進行）
- 🟡 黄（注意）
- → 繰り返し

---

## 🛠 使用技術
- デジタル出力制御
- 時間管理（millis または delay）
- 状態遷移（state管理）

---

## 💡 工夫した点

- 状態（state）ごとに処理を分けて可読性を向上
- 時間を変数として管理し調整しやすく設計
- 複数LEDの制御を整理して実装

---

## 🔧 使用部品

- ELEGOO Uno R3
- LED（赤・黄・緑）
- 抵抗 ×3
- ブレッドボード
